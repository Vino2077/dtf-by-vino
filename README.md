# DTF by Vino

Альтернативный неофициальный клиент для [DTF](https://dtf.ru) на Flutter. Весь основной функционал сайта плюс фишки, которых нет в оригинале.

> ⚠️ Неофициальное приложение. Не связано с DTF / Osnova. Работает поверх публично доступного API сайта.

## 📥 Скачать

**[→ Скачать последнюю версию (APK)](https://github.com/Vino2077/dtf-by-vino/releases/latest)**

Android: скачай APK из раздела Releases и установи (может понадобиться разрешить установку из неизвестных источников).

## ✨ Возможности

- **Лента**: Популярное, Свежее, Новости редакции, Моя лента. Реакции и переход к комментариям прямо из ленты.
- **Реакции** настоящими картинками (не эмодзи), с анимацией; сетка сортируется по частоте использования — любимые сверху.
- **Комментарии**: древовидные ветки, вложения (фото/видео/GIF), реакции, заметки об авторах.
- **Чтение удалённых комментариев** и история их правок (через архив сообщества).
- **Редактор постов**: блоки (текст, заголовки, цитаты, списки, разделители, медиа, аудио), публикация.
- **Уведомления** с переходом к нужному комментарию и счётчиком на колокольчике.
- **Поиск**: топ блогов и популярные комментарии.
- **Медиа**: видео со звуком, полноэкранный просмотр, анимированные аватарки.
- **Кастомизация**: акцентный цвет, свой фон, чёрная AMOLED-тема.

## 🛠 Сборка из исходников

Нужен установленный [Flutter SDK](https://docs.flutter.dev/get-started/install).

```bash
git clone https://github.com/Vino2077/dtf-by-vino.git
cd dtf-by-vino
flutter pub get
flutter build apk --release
# готовый APK: build/app/outputs/flutter-apk/app-release.apk
```

Для поиска GIF используется [GIPHY API](https://developers.giphy.com/) — ключ уже вшит в код (`lib/services/giphy_service.dart`); при желании замени на свой бесплатный.

## 🧩 Технологии

Flutter · Dart · Provider · video_player · cached_network_image. Обращения к API DTF/Osnova — реверс-инжиниринг публичных эндпоинтов (подробные заметки в комментариях `lib/api/dtf_api.dart`).
